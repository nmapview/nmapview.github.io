From overview to deep dive

Move from broad asset visibility to detailed host investigation in a few clicks.

1. Host Overview

Inventory hosts, operating systems, exposure levels and rarity scores. Visualise exposure with Open Ports Per Host and Operating System Distribution charts.

2. Open Services

Review exposed services, versions, certificates, HTTP metadata, CPEs and vulnerability references. Service Distribution Across Hosts highlights common and unusual services.

3. Service Summary

Group hosts by service, product, version and port. Explore Host-Service Relationships and Service-Port Heatmaps to identify common baselines and unusual deployments.

4. Host Details

Drill into ports, banners, scripts and complete NSE output for individual hosts. Use the Host-Port Matrix to investigate exposure patterns host by host.

Built for the full workflow

From jump-box triage to polished client deliverables. The same HTML file works as a pivot point, a handoff artifact, and a prompt surface for LLM analysis.

🏢
Internal Network Assessments
Enumerate hosts, spot version drift, and document findings in a portable format — without a reporting pipeline on the jump box.
🌍
External Attack Surface Reviews
Map public-facing infrastructure, filter to SSL endpoints, and track remediation progress across periodic scans.
📝
Pentest Reporting
Hand clients an interactive report as a single file. No account, no server, no expiry — scope it to the hosts that matter first.
🔗
Downstream Tool Handoff
Scope and filter, then export to CSV or JSON for httpx, nuclei, testssl.sh, or ssh-audit follow-up.
🤖
LLM Analysis
The exported CSV or JSON from a scoped report is a better prompt surface than raw Nmap XML — grouped, labeled, and easy to slice.
📦
Asset Inventory
Build a living inventory from periodic scans. Export to Excel for management review, compliance tracking, or audit handoff.
