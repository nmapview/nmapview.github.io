NmapView – Nmap XML Report Viewer

Transform raw Nmap XML into a single portable interactive HTML report.

No backend. No database. No reporting pipeline.
Generate a report and open it anywhere.

View on GitHub Live Demo Download XSL

Why NmapView?

Portable

Everything lives in a single HTML file.

Interactive

Search, sort, filter, highlight and export instantly.

No Infrastructure

No backend, database, API or services required.

Built for Triage

Quickly identify exposed services, versions and anomalies.

Quick Start

curl -fsSL -o NmapView.xsl \
https://github.com/dreizehnutters/NmapView/releases/latest/download/NmapView.xsl

xsltproc -o report.html NmapView.xsl scan.xml

Open report.html in your browser.

Features

  • Single-file HTML reports
  • Host overview and asset inventory
  • Service grouping and version analysis
  • CSV, Excel and JSON export
  • Global keyword highlighting
  • Interactive visualizations
  • NSE output review
  • Host and service relationship analysis

Use Cases

  • Internal network assessments
  • External attack surface reviews
